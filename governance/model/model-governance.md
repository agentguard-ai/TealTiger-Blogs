# Model Governance and Control

Not all models are equal. TealTiger introduces **explicit model governance**.

## What Is Governed

- Which models are allowed
- For which task categories
- Under what risk classification

## Deterministic Model Selection

Model usage is driven by governance contracts, not runtime heuristics.

## Why This Matters

Uncontrolled model switching introduces:
- Compliance risk
- Cost volatility
- Inconsistent behavior

## Outcome

Model governance delivers predictability, traceability, and operational safety.
